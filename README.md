# Car Sales System

A C-based car sales management system for **Ansh Garage**. The program allows users to view available car stock, buy cars, record sales, apply discounts, calculate finance payments, and collect customer feedback.

## Features

- View available cars, including model, price, year, and stock
- Sort cars by year in descending order
- Buy one or more cars in a single purchase
- Prevent customers under 18 from buying cars
- Apply a 10% discount for customers over 60
- Choose between upfront payment and finance payment
- Calculate monthly finance payments
- Store sales data in a binary file
- View previous sales records
- Collect and display customer feedback

## Technologies Used

- C programming language
- Standard C libraries:
  - `stdio.h`
  - `stdlib.h`
  - `string.h`
  - `time.h`
  - `math.h`

## Files

| File | Description |
|---|---|
| `Carsales.c` | Main C source code for the car sales system |
| `sales_data.bin` | Binary file created automatically to store sales data |

> Note: `sales_data.bin` is generated by the program when sales data is saved. You usually do not need to upload this file to GitHub.

## How to Compile

Open a terminal in the project folder and run:

```bash
gcc Carsales.c -o carsales -lm
```

The `-lm` flag is needed because the program uses the `pow()` function from the math library.

## How to Run

After compiling, run:

```bash
./carsales
```

On Windows, you may need to run:

```bash
carsales.exe
```

## Menu Options

When the program starts, it displays the following menu:

```text
--- Car Sales Menu ---
1. View Cars Stock
2. Buy Cars
3. View Sales Data
4. Customer Feedback
5. Exit
```

## Example Cars Included

The system starts with example stock for:

- Toyota Corolla
- Honda Civic
- Ford Focus
- BMW 3 Series

## Finance Payment

If the customer chooses finance, the program asks for:

- Finance duration in months
- Annual interest rate

It then calculates and displays:

- Total finance amount
- Monthly payment
- Finance duration

## Customer Feedback

After a successful purchase, the customer can leave:

- A rating from 1 to 5
- A feedback comment
